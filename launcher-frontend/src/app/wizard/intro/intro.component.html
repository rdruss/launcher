Launchpad Project Wizard

Create functional applications (quickstarts) or project frameworks that enable you to quickly develop cloud-native applications using the Eclipse Vert.x, WildFly Swarm, or Spring Boot runtimes. You can:

  • Select an existing fully functional quickstart or create a runtime project framework.
  • Import the quickstart or runtime project into JBoss Developer Studio (or your preferred IDE). Then adapt, run, and test it locally.
  • Deploy your completed project to Red Hat OpenShift Online.

For more information, see: Release Notes and Supported Runtime Versions.
