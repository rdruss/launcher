Microservices Starter & Quickstart

This DevTools will let you as a developer to create a Java project, that next you can import within the JBoss Developer Studio. It proposes two opinionated ways to achieve this goal either using one of our QuickStarts supporting different Microservices patterns as REST, Secured REST, Health Check or either by using one of the starters proposed as Eclipse Vert.x, Wildfly Swarm or Spring Boot. The different projects support to be launched locally within a terminal of your machine or deployed on top of an OpenShift Container Platform installation

Learn more about this on the documentation.
